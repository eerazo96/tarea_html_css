# Ejercicio HTML y CSS
 
Este repositorio contiene un pequeño proyecto de práctica con archivos:
 
- `index.html` para la estructura de la página.

- `styles.css` para los estilos.

- Carpeta `img/` con imágenes usadas.
 
Es un ejercicio sencillo para aprender a trabajar con HTML y CSS y subirlo a GitHub.
